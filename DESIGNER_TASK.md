# Задача дизайнеру (через CTO Invoker)

## Цель
Сделать игру-подобную 2D визуализацию офиса агентов, где видно:
- личные комнаты с кроватями,
- общий офис,
- кухню для случайных встреч,
- социальные связи (дружба/коллеги).

## Что отдать в дизайн
1. **UI-kit сцены**: цвета, типографика, карточки зон, стили агентов.
2. **Спрайтовый стиль**: 8 агентов (idle/walk/talk/work/rest).
3. **Анимации переходов**: room → office → kitchen (плавно, 250–400ms).
4. **Система отношений**: визуальные маркеры (иконки/линии/подсветка).
5. **HUD**: текущая стадия pipeline + события дня.

## Ограничения
- Визуалка должна быть чистой, без перегруза.
- Мобилка: обязательно читаемость на узком экране.
- Должно выглядеть как мини-игра, а не статичный дашборд.

## Definition of Done
- Figma/макеты + токены стиля + список ассетов.
- Состояния агентов описаны и переданы в dev.
- Сценарий кухни (встречи/обмен опытом) визуально отражён.
