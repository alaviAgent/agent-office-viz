# CTO Invoker — детальный план и распределение

## Что правим немедленно
1. Живое перемещение (без телепорта) по маршрутам: комната → офис → кухня → комната.
2. Облака общения над персонажами при синках/ивентах.
3. Случайность поведения: наличие задач, состав кухонной группы, реплики.
4. Цикл симуляции по времени: проверка задач + кухонные встречи каждые 10 минут.

## Раздача по агентам
- **Chen (PM):** оформить сценарии поведения (state machine + события) и acceptance criteria.
- **Designer:** финальный пиксельный арт-пак (персонажи, мебель, тайлы пола, UI bubbles).
- **Tinker (Dev):** система маршрутов/анимации движения + рендер сцен.
- **Oracle (DS):** веса случайности (task probability / social interactions).
- **Io (MLOps):** стабильный хостинг + healthcheck URL.
- **QA:** 10+ сценариев (движение, кухня, облака, возвраты, пустой бэклог).

## Definition of Done
- Визуально видно, что агенты именно ИДУТ.
- Облака общения появляются по событиям.
- Кухня активируется каждые 10 минут сим-времени.
- Симуляция не застревает и не дёргается.
- Демо URL стабилен и доступен по Tailscale.
